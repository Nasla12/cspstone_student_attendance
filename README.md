# Attendance Shortage Alert — Capstone Project (BCA)

A console-based Python project for BCA students. Build a simple **Student Record & Attendance Alert System** using variables, input, conditions, loops, functions, and file handling.

## What You Will Build

- **Option 1** — Add student records to a text file (one student per line)
- **Option 2** — Search a student by roll number and show attendance report
- **Functions** — `calculate_average()` and `check_shortage()` for attendance logic
- **Escape keyword** — Type `done` during entry to return to the main menu

## Quick Start

1. **Fork** this repository on GitHub (see [STUDENT_GUIDE.md](docs/STUDENT_GUIDE.md))

3. Open `assignment/STUDENT_INFO.txt` and fill in your details

5. Complete `assignment/attendance_alert.py` — the file has empty function placeholders only
6. **Commit and push** your finished work to your fork

## Record Format (students.txt)

Each student is stored on **one line**, fields separated by spaces:

```
<RollNo> <Name> <Stream> <Subject1:Percentage> <Subject2:Percentage> ...
```

Example:

```
R01 Arjun BCA English:60 Maths:72 Python:85 DBMS:55 OS:78
```
